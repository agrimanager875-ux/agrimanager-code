# CycleGym Price-Regime OOD

This directory contains the non-smoke scripts for the CycleGym price-regime
OOD experiment.

The implemented setup matches the intended first paper version:

- 19-year crop-planning episodes
- two candidate crops:
  - `CornRM.100`
  - `SoybeanMG.3`
- train on variable but non-dominant in-distribution price samples for a
  realistic Corn Belt corn-soybean rotation setting
- validate during training on paired held-out scenarios across three price regimes:
  - `id_test`
  - `high_corn`
  - `high_soybean`

The validation split is explicitly paired: for the same weather, location, year
window, and seed, the only thing that changes across the three paired rows is
the crop-price vector. Each parquet row stores:

- `price_regime`
- `paired_scenario_id`
- `paired_base_crop_prices`

Each validation row preserves the full environment config, so training-time
validation can group episodes by regime and paired scenario. Training scripts
set `data.validation_axis=price_regime`, so the experiment tracker reports separate validation
curves for the ID and OOD price regimes every 10 steps.

## Dataset Scale

- train: 1600 scenarios
- validation: 384 episodes
  - 128 paired scenarios
  - 3 regimes per paired scenario
- test: not generated by default

## Run Order

1. Build datasets on CPU:

   ```bash
   bash experiments/cycles_gym_price_regime/run_build_datasets.sh
   ```

2. Train LLM think / no-think on GPU:

   ```bash
   bash experiments/cycles_gym_price_regime/run_llm_train_price_regime_think.sh
   bash experiments/cycles_gym_price_regime/run_llm_train_price_regime_no_think.sh
   ```

3. Train the NN baseline:

   ```bash
   bash experiments/cycles_gym_price_regime/run_nn_train_price_regime_n1.sh
   bash experiments/cycles_gym_price_regime/run_nn_train_price_regime_n8.sh
   ```

The reporting path is the validation curve logged during training. There is no
separate post-training evaluation step for this experiment because `id_test`,
`high_corn`, and `high_soybean` are validation regimes, not test regimes.
