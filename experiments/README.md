# Experiments

This directory stores runnable experiment definitions and local outputs. The
directory names now follow the V2 paper-facing experiment-card structure when a
V2 card exists.

## Dataset Sources

See [`wofost_gym_datasets.md`](wofost_gym_datasets.md) for the centralized
definition of the WOFOST-Gym weather-regime, 20-crop, and maize weather pools.

## Paper Experiment Map

The table maps the paper-facing settings to artifact directories and runnable
entrypoints. Training scripts are fixed definitions for the listed setting.
Evaluation uses the shared `entrypoints/eval/eval.sh` or
`entrypoints/eval/nn_eval.sh` on the validation parquet/checkpoint generated by
the corresponding training script unless a setting has a dedicated eval wrapper.
